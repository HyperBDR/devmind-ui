# E2E test reports

Reports generated by the **devmind-browser-e2e** Skill (Browser MCP) are saved here. This directory lives in the **devmind-ui** project.

## Naming convention

- **Filename**: `e2e-report_<commit_id>.md`
- **`<commit_id>`**: First 6 characters of the Git commit at run time (e.g. `e2e-report_abc123.md`). If not in a Git repo or `git rev-parse --short=6 HEAD` fails, the report may be named `e2e-report_unknown.md` or `e2e-report_YYYYMMDD-HHMM.md`.

Reports are written by the skill after each run; this directory is the single place for E2E report artifacts.
